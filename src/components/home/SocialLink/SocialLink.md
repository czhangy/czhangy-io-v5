# SocialLink

A square icon button that links to an external social profile, opening in a new tab.

## Props

| Prop   | Type              | Required | Default | Description                      |
| ------ | ----------------- | -------- | ------- | -------------------------------- |
| `href` | `string`          | Yes      | —       | URL to link to                   |
| `icon` | `React.ReactNode` | Yes      | —       | Icon to render inside the button |
