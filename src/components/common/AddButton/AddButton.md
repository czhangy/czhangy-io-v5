# AddButton

The submit button row at the bottom of a modal form.

## Props

| Prop       | Type         | Required | Default | Description                              |
| ---------- | ------------ | -------- | ------- | ---------------------------------------- |
| `label`    | `string`     | Yes      | —       | Button text (e.g. "Add", "Save")         |
| `disabled` | `boolean`    | Yes      | —       | Whether the submit button is disabled    |
| `onSubmit` | `() => void` | Yes      | —       | Called when the submit button is clicked |
