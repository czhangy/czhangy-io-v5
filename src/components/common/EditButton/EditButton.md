# EditButton

A minimal button wrapping `EditIcon`. Consumers supply visual styles via `className`.

## Props

| Prop        | Type         | Required | Default | Description                          |
| ----------- | ------------ | -------- | ------- | ------------------------------------ |
| `onClick`   | `() => void` | Yes      | —       | Click handler                        |
| `className` | `string`     | No       | —       | Additional class for consumer styles |
