# ReadingPanelLoader

Entry point for the Reading status panel. Passes the READING label and BookIcon down to ReadingPanel.

## Props

| Prop   | Type     | Required | Default | Description      |
| ------ | -------- | -------- | ------- | ---------------- |
| `cols` | `number` | Yes      | —       | Grid column span |
| `rows` | `number` | No       | `1`     | Grid row span    |
