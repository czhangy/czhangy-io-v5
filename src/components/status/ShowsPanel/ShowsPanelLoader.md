# ShowsPanelLoader

Async server component that fetches the current show entries from the database and their metadata from the TVmaze API, then renders ShowsPanel with the resolved data.

## Props

| Prop        | Type     | Required | Default | Description             |
| ----------- | -------- | -------- | ------- | ----------------------- |
| `className` | `string` | No       | —       | Forwarded to ShowsPanel |
