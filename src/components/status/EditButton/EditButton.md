# EditButton

An icon button used in status panel headers to trigger edit mode.

## Props

| Prop       | Type         | Required | Default     | Description                               |
| ---------- | ------------ | -------- | ----------- | ----------------------------------------- |
| `onClick`  | `() => void` | No       | `undefined` | Callback fired when the button is clicked |
| `disabled` | `boolean`    | No       | `undefined` | Disables the button when true             |
