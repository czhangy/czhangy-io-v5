# VickyPanel

A status panel displaying a square photo of Vicky.

## Props

| Prop   | Type     | Required | Default | Description                            |
| ------ | -------- | -------- | ------- | -------------------------------------- |
| `cols` | `number` | Yes      | —       | Number of grid columns the panel spans |
| `rows` | `number` | No       | `1`     | Number of grid rows the panel spans    |
